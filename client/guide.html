Gameplay Tutorial

First, please input two users' names.

Starting from 0, each of the user type a natural number which is minimally lager than the last
number by 1, or maximally larger that the former one by 3. In level one, whoever gets to
10 will be the winner. In level 2 and 3, who reaches 19 and 29 respectively will be the winner.

Below are the examples:

    Possibility of the first 3 inputs
  • User 1: 0
    1. User 2: 1
      1. User 1: 2
      2. User 1: 3
      3. User 1: 4
    2. User 2: 2
      1. User 1: 3
      2. User 1: 4
      3. User 1: 5
    3. User 2: 3
      1. User 1: 4
      2. User 1: 5
      3. User 1: 6
    Game example - Level 1:
  • User 1: 0
  • User 2: 2
  • User 1: 3
  • User 2: 4
  • User 1: 6
  • User 2: 7
  • User 1: 10
  • User 1 won!
